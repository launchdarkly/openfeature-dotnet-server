Class ProviderConfiguration

Configuration options for Provider. This class should normally be constructed with Builder().

Inheritance

ProviderConfiguration

Remarks

Instances of ProviderConfiguration are immutable once created. They can be created using a builder pattern with Builder().

Remarks

The ProviderConfigurationBuilder has methods for setting any number of properties, after which you call Build() to get the resulting ProviderConfiguration instance.

Examples
    var config = ProviderConfigurationBuilder.Builder()
        .Logging(Components.NoLogging)
        .Build();
