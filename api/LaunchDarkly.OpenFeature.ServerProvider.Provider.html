    GetClient()

    Get the underlying client instance.

    If the initialization/shutdown features of OpenFeature are being used, then the returned client instance may be shutdown if the provider has been removed from the OpenFeature API instance.

    This client instance can be used to take advantage of features which are not part of OpenFeature. For instance using migration flags.

    GetStatus()

    Get the status of the provider.

    Declaration
    public override ProviderStatus GetStatus()
    Returns
    Type Description
    ProviderStatus

    The current OpenFeature.Constant.ProviderStatus
    Overrides
    OpenFeature.FeatureProvider.GetStatus()
    Remarks

    If a provider does not override this method, then its status will be assumed to be OpenFeature.Constant.ProviderStatus.Ready. If a provider implements this method, and supports initialization, then it should start in the OpenFeature.Constant.ProviderStatus.NotReadystatus . If the status is OpenFeature.Constant.ProviderStatus.NotReady, then the Api will call the OpenFeature.FeatureProvider.Initialize(OpenFeature.Model.EvaluationContext) when the provider is set.
